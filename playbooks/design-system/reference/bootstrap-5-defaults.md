# Bootstrap 5 Defaults Reference

Quick reference for key Bootstrap 5 defaults. The audit should extract actual values from `node_modules` rather than relying solely on this reference.

## Colours

| Name | Default |
|------|---------|
| Primary | `#0d6efd` |
| Secondary | `#6c757d` |
| Success | `#198754` |
| Danger | `#dc3545` |
| Warning | `#ffc107` |
| Info | `#0dcaf0` |
| Light | `#f8f9fa` |
| Dark | `#212529` |

## Type Scale

| Class | Size |
|-------|------|
| `fs-1` | 2.5rem |
| `fs-2` | 2rem |
| `fs-3` | 1.75rem |
| `fs-4` | 1.5rem |
| `fs-5` | 1.25rem |
| `fs-6` | 1rem |
| `small` | 0.875em |

## Spacing Scale

| Level | Value |
|-------|-------|
| 0 | 0 |
| 1 | 0.25rem |
| 2 | 0.5rem |
| 3 | 1rem |

| 5 | 3rem |

## Breakpoints

| Name | Min-width |
|------|-----------|
| sm | 576px |
| md | 768px |
| lg | 992px |
| xl | 1200px |
| xxl | 1400px |

## Other Defaults

- Base font size: `1rem` (16px)
- Base line-height: `1.5`
- Font family: system font stack
- Border radius: `0.375rem`
- Box shadow (sm): `0 .125rem .25rem rgba(0,0,0,.075)`
